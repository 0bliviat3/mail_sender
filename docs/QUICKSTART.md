# 빠른 시작 가이드

## 두 가지 버전

### 1. 기본 버전 (email_sender.py)
- 간단하고 사용하기 쉬움
- 이름 변수만 지원 ({{이름}})
- 초보자에게 추천

**실행 방법:**
```bash
python email_sender.py
```

### 2. 고급 버전 (email_sender_advanced.py)
- 모든 Excel 컬럼을 변수로 사용 가능
- HTML 메일 지원
- 메일 간 대기 시간 설정
- 메일 미리보기 기능
- 고급 사용자에게 추천

**실행 방법:**
```bash
python email_sender_advanced.py
```

## 10분 안에 시작하기

### Step 1: 필수 프로그램 설치 (5분)
1. Python 설치: https://www.python.org/downloads/
2. 명령 프롬프트(CMD)를 열고:
```bash
pip install pandas openpyxl
```

### Step 2: Gmail 앱 비밀번호 생성 (3분)
1. Google 계정 관리 → 보안
2. 2단계 인증 켜기
3. 앱 비밀번호 생성 (16자리)
4. 복사해두기

### Step 3: 프로그램 실행 (1분)
```bash
python email_sender.py
```

### Step 4: 샘플로 테스트 (1분)
1. "샘플 파일 생성" 클릭
2. 생성된 Excel 파일 열어서 자신의 이메일 주소 입력
3. 파일 저장 후 프로그램에서 불러오기
4. 발신자 정보 입력
5. "테스트 모드" 체크된 상태로 "메일 전송 시작"

## 실제 사용 시나리오

### 시나리오 1: 직원들에게 급여명세서 발송

**Excel 파일:**
| 이메일 | 이름 | 첨부파일 |
|--------|------|----------|
| kim@company.com | 김철수 | C:/급여명세서/김철수_2024_11.pdf |
| lee@company.com | 이영희 | C:/급여명세서/이영희_2024_11.pdf |

**메일 내용:**
```
안녕하세요, {{이름}}님

2024년 11월 급여명세서를 첨부파일로 보내드립니다.
확인 후 문의사항이 있으시면 연락 주시기 바랍니다.

감사합니다.
인사팀
```

### 시나리오 2: 고객에게 맞춤 안내 메일 (고급 버전)

**Excel 파일:**
| 이메일 | 이름 | 회사명 | 담당자 | 계약일 |
|--------|------|--------|--------|--------|
| a@example.com | 홍길동 | A회사 | 김대리 | 2024-12-01 |
| b@example.com | 이철수 | B회사 | 박과장 | 2024-12-15 |

**메일 내용 (고급 버전):**
```
안녕하세요, {{회사명}} {{이름}}님

귀사의 담당자 {{담당자}}님과 논의된 내용에 따라
{{계약일}}부터 서비스가 시작됩니다.

자세한 내용은 첨부파일을 참고해주세요.

감사합니다.
```

## 트러블슈팅

### Q: "인증 실패" 오류가 나요
A: Gmail 앱 비밀번호(16자리)를 사용하세요. 일반 비밀번호는 안됩니다.

### Q: 한글이 깨져요
A: Excel을 UTF-8로 저장하거나, CSV는 "UTF-8 BOM"으로 저장하세요.

### Q: 첨부파일이 안 보내져요
A: 
- "첨부파일 사용" 체크박스 확인
- Excel의 '첨부파일' 컬럼명 확인
- 파일 경로가 정확한지 확인

### Q: 너무 많은 메일을 보내면?
A: 
- Gmail: 하루 500통 제한
- 대기 시간을 1-2초로 설정
- 여러 날에 걸쳐 분할 발송

### Q: 스팸으로 분류되는 것 같아요
A:
- 대기 시간 늘리기 (3-5초)
- 하루 발송량 줄이기 (100-200통)
- 메일 내용을 더 정중하게 작성

## 고급 사용 팁

### 1. 여러 변수 사용 (고급 버전)
Excel에 원하는 컬럼 추가하고 {{컬럼명}}으로 사용

### 2. HTML 메일 (고급 버전)
"HTML 형식 사용" 체크 후:
```html
<h2>안녕하세요, {{이름}}님</h2>
<p><b>중요:</b> 첨부파일을 확인해주세요.</p>
<p style="color: blue;">감사합니다.</p>
```

### 3. 대량 발송 분할
Excel 파일을 날짜별로 나누어:
- 월요일: 1-500명
- 화요일: 501-1000명
- 수요일: 1001-1500명

### 4. 테스트는 필수!
실제 발송 전 항상:
1. 테스트 모드로 확인
2. 자신에게 먼저 발송
3. 내용 확인 후 실제 발송
